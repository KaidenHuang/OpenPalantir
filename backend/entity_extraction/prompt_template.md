# 实体识别任务

## 实体类型
- person: 人名
- organization: 组织（公司/机构/学校）
- location: 地点（国家/城市/地理特征）
- event: 事件（会议/战争/活动）
- concept: 抽象概念（政策/理论/思想等）
- other: 其他实体类型 （如 动物、植物、产品等）

## 输出JSON格式
```json
[{"n":"名称","t":"类型","c":次数,"bn":"别名","d":"对该实体在文本中的简要描述（身份、角色、关键属性等）"}]
```

## 规则
1. 名称用标准全名，类型限上方5种值
2. `c`为int（出现次数），`bn`可选（别名）
3. `d`为字符串，简要描述该实体在文本中的角色、身份或关键特征（50字以内），无明确信息时可省略
4. 只返回纯JSON数组，无额外文本
5. 语言与输入文本一致
6. 相同名称+类型的实体去重

## 输入
{text}
